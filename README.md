# social_data_analysis
### 소셜데이터로 본 방한 외국인의 방문지 분석

[분석 목적]

기존에 잘 알려진 관광 명소가 아닌, 실제 외국인 관광객들이 관심을 가지고 방문하는 장소를 소셜미디어 데이터를 기반으로 파악하여 외국인의 실질적인 관광 선호도와 방문 트렌드를 분석하고자 함
